# Ontology-for-Robotics
Robot Structure Design using Ontologies
The ontology included for this application relies on the IEEE-ORA Standard released in 2015.
It contains:
* Editor and Parser for SUO/KIF Ontologies
* Rule extraction and documentation
* Graphical robot designer with three modes
  * Configuration
  * Skill
  * Free
To execute just double/click in the jar file or execute in the command line>
$ java -jar TheRobotBuilder.jar
